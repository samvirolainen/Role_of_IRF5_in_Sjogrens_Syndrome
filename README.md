# Colocalization of Autoimmune Risk Loci with Quantitative Trait Loci (QTLs) -

## Project Goals 
- Curate Genome Wide Association Study (GWAS) data for autoimmune diseases and fine map risk loci to identify causal variants 
- Identify and fine map QTL data 
- Perform colocalization analysis to identify the impact of genetic variation on mRNA (eQTL) and protein abundance (pQTL)

## Data Availability
Raw and processed data from the publication are **not distributed** in this repository due to licensing and size restrictions.  
Users must obtain data directly from the original publication or associated repositories.

A reproducible Conda environment for this project will be included in once analyses commence.
